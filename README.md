# MMORPG Server (C#)

C#으로 **MMORPG 서버를 처음부터 직접 구현하며 학습한 내용을 기록하는 프로젝트**입니다.  
단순 기능 구현을 넘어 서버 아키텍처, 네트워크 통신, 멀티스레딩, 확장성에 초점을 두고 있습니다.  
추후에는 게임 기획부터 설계, 배포까지 전 과정을 직접 진행할 예정입니다.



## 🔖 목표

* MMORPG 서버의 전체 흐름 이해
* C# 기반 네트워크 프로그래밍 학습
* 멀티 클라이언트 환경에서의 동시성 처리
* 실제 서비스 서버를 고려한 구조 설계 연습
